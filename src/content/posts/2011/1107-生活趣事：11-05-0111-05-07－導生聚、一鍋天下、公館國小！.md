# 5/1(日)
今天早上主日聚會，晨興聖言要換新書了～而且這次我除了買中文版，還買了英文版呀！XD 下午是家教，然後領錢了～XD 我跑去買了Portal2，然後也幫河水(后羿射月亮)買了一套，一起玩雙人模式XD

# 5/2(一)
今天早上英文課，老師因為我們都沒做出有聲書成品感到很生氣，他說下禮拜一定要交，不然就零分！我和我們組員就在英文課下課留下來錄音，錄完再趕去貓咪果醬，嗯...戰鬥系統一直都沒動作啊Orz...

下午是程式設計，上完後就跑去本部的英語自學教室湊時數，順便去樸102拿有聲書比賽的報名表，晚上和浩庭去吃本部開的拉麵店，還不錯XD 映竹因為手機沒電就沒辦法約來跟我們一起來吃了QQ~

晚上是Vocal班，教了樂理，聽都聽不太懂呀XD""" 努力搞動那些和弦、調......等等東西。XD

# 5/3(二)
早上去數位邏輯，中午原本要去邵弟兄家讀經小組，但教官要跟我和映竹講一些事情，我們就到新教官室(女二舍的一個小房間)去找教官，之後就跟著教官到男三舍的一個小房間在看教官研究的硬體槍結合軟體做模擬射擊，預計禮拜四早上要去公館國小宣教，我們就決定請公假一起去XD 下午就跑去聽離散數學錄影，這是最後一次錄影實驗XD 下禮拜就沒有了～=D

# 5/4(三)
早上去弟兄之家晨興，接著就去上自然探索，原本今天要口頭報告，但延至下禮拜XD 雖然說我已經準備好了啦XDDDDD 中午趕往分部參加導生聚，這次的導生聚弄的像是生日派對一樣，實際上也是啦XD 很開心可以吃到一堆PizzaXD 然後突破The Cake is a lie.的謠言XDDDDDDD 下午是程式設計，晚上就跟安偉、博懷、浩庭、毓偉、晉嘉去吃位於板橋的一鍋天下，是一家吃到飽的店XD 但結果我吃鴨血好像比吃肉還多XD"""" 真糟～

# 5/5(四)
早上8點到教官室找教官，然後接著教官就開車載我們到公館國小，公館國小是個很自然的小學呢XD 我們在五年甲班幫忙給小朋友做實彈射擊模擬，順便教導他們槍是啥東西XD 小朋友很熱情、很可愛呢>////< 中午回來趕快吃完飯就去微積分教室，上完微積分後就回家搞數邏實驗的預報和結報，每禮拜四的例行公事呢！XD"""

# 5/6(五)
早上國文上課，接著離散數學上課，中午趕去本部將做好的有聲書交去樸102，交完就去逛了一下師大蘋果店，逛完就回分部，去PC室休息，PC室大一的只有威豪和維倫，就在那邊待了一下，然後就去上數邏和數邏實驗，這次實驗比較久，因為我Pin不小心打錯Orz...結果造成無法挽回的悲劇(好啦，實際上也沒那麼慘啦XD"") 晚上因為太累就回家休息了ˊˋ 就沒去教會ˊˋ...

# 5/7(六)
下午是ezgo開發聚會，我就跑去吃東西XDDDDDDDDDD 我整場幾乎沒幫游力上什麼忙，都在吃免錢午餐和晚餐XD""""

不知道這樣我下次還可不可以去XD~
