# 7/9(六)
早上首先就和表哥以及我媽到附近的咖啡店吃早餐，接著就到北車去搭車去福隆(實際上是澳底)找親戚。今天也正好我們系上的人要去福隆場勘，我們就坐了同一班火車去。到了福隆後，我們就搭免費接駁車到澳底，在那邊吃了午餐，就到了親戚家。結果我們在那邊幾乎都在玩我的筆電XD"" 晚上就坐火車回來了，還蠻累的Orz... 大概也是因為跟音樂祭衝突，所以超多人Orz... 超擠Orz... 不過也是因為這樣才會有免費接駁車呀=P

# 7/10(日)
今天帶我表哥去惡搞天使家，我們就去逛了三和夜市，然後我就順便去買衣服，到了Hang Ten，我買了一套衣服(一件襯衫和一件牛仔褲)，看起來還不錯呀！下次決定來用頭髮=w=

# 7/11(一)
早上一大早就衝往昆陽站，然後走到南港高中和慧君老師見面，我們就到了南港高中的電腦教室，在那邊寫職棒的那隻程式，就這樣從早上寫到了下午，寫的差不多後，就回去了～XD 今天也是蠻累的一天呀Orz...

# 7/12(二)
今天去逛了微風廣場，最主要是我表哥想去MUJI無印良品買包包，順便還逛了上面的書店，我買了一本「宅男時尚寶典」來做為追求時尚的入門書，這本書感覺真的很不錯呢！晚上就在微風B1F吃，吃了涮涮鍋，還不錯吃XD 吃完就回家去了～

# 7/13(三)
早上去師大人發系系辦家教，家教完後就先回家。晚上六點和我表哥以及聖別、璋駿以及啟源吃飯，吃完飯後還去逛了光華商場。接著我表哥要和他國小同學去吃飯，我就送他到國父紀念館站就回來。據說他們喝酒喝到最後有人在捷運站內吐了，喝酒不可以喝得太兇呀Orz...

# 7/14(四)
早上去師大人發系系辦家教，家教完後就回家了。之後就宅了Orz......

# 7/15(五)
今天在家學會了outlet的意思，本來想約河水和惡搞天使到公館的一家outlet店去逛，但是大家下午都睡著了(死) 最後我就和我表哥以及他的國小朋友去西門站的熊一吃燒烤，大家吃的很開心，吃到他們快打烊就回家去了XD
