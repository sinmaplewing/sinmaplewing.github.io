從題目即可看出f91(N)=N-10，當N&gt;=101。那麼當N&lt;=100真的要下去遞迴看看嗎？首先可以先手算出當90&lt;=N&lt;=100：
```
f91(90) = f91(f91(101))
= f91(91) = f91(f91(102))
= f91(92) = f91(f91(103))
= f91(93) = f91(f91(104))
= ......
= f91(99) = f91(f91(105))
= f91(100) = f91(f91(111))
= f91(101) = 91
```
剩下的數值即可利用下列式子證明出1&lt;=N&lt;90都會等於91：
```
  f91(N) = f91(f91(N+11))
= ... = f91(f91(f91(...f91(a)))) // 90 <= a <= 100
= f91(f91(f91(...f91(91))))
= 91
```

}
