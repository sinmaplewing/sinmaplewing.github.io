# 6/19(日)
早上沒去主日聚會，中午也沒去初信成全，倒是下午的家教有去，將高中最後機率與統計給它弄掉這樣=w= 講得都是信心水準與信賴區間，大概就這樣吧！？

# 6/20(一)
今早上完離散，為毓偉與浩庭上數位邏輯，上得差不多後，下午就去考程式設計上機，我雖然很快就寫完，但我一直不敢舉手出去，因為我大概在助教講完題目解說後，沒多久就完成了，之後我就是一直檢查這樣Orz... 考完後，又繼續把數位邏輯搞定。這樣明天應該沒問題吧！

# 6/21(二)
今早考數邏，我覺得是歷年最簡單的一次期末考！但是看來其他人不是這麼想Orz... 後來我認為是：有上課的人會覺得是歷年來最簡單的，但沒上課的人會發現題型與考古題完全不一樣Orz... 中午和威豪到C208去檢驗他的電路，弄得差不多就去吃飯。下午就繼續幫毓偉與浩庭搞定他們的數邏作業，考完試再弄作業的感覺很不一樣呀=w=

# 6/22(三)
早上是最後一節自然探索，看到很多楊恩生教授拍的影片，就這樣結束了這門課。這門課讓我對生物有了不一樣的體驗，感覺很棒的一門通識。下午到PC室去拿離散考卷，109分！只被扣一分，原因是因為我沒仔細看到它要我求的是probability Orz... 之後就先跟小光和微米到系辦二樓上學期拿來當打掃休息間的那間教室念書，我都不知道這間教室可以隨意使用！！真的太棒了！等到毓偉來找我後，我們就到圖書館外面的桌椅念微積分，但是蚊子太多，就跑到學七舍外面的桌椅念，等到浩庭和志中來了後，就決定去找間教室念書，因此就用了C209，用到有學長姊進來就離開，我們就去吃晚餐，吃完就跑到PC室繼續念，還遇到園長和少膺也在念微積分，之後大家就一起念，念完就結束今天了！

# 6/23(四)
今天一早去了邏輯概論的教室，裡面沒人，所以就去了PC室念微積分，正好園長和少膺也在那裡，我們唸到中午，就去吃午餐～吃完再到微積分教室念了一下，念完就考了。考完後，我們貓咪果醬的一群人就要去晚上聚餐了，我本來沒要去的，但是後來還是跟去了，我們到了位於忠孝復興站附近的BROADWAY百老匯餐廳，這家餐廳價位還蠻高的，東西也還不錯，但是等待花的時間太久了= = 一杯冰咖啡要等到兩個鐘頭Orz...(從這次聚會以後，園長又多了香蕉園園長的暱稱，完全是因為他因為等待他的餐點過長(真的很長)，然後玩弄浩庭的香蕉...不對...是浩庭所吃的冰點裡的香蕉。XD) 結束後，和Psi Omega回到忠孝新生，逛了一下光華，還有一家隱藏在我們家附近的店，之後就各自回家了～

# 6/24(五)
早上是最後的考試：英文與國文，考完後趕校車到凱莉準備吃導生聚XD 然後我們貓咪果醬的坐一桌，又開始聊起昨天晚上百老匯餐廳的事。然後我說了一句「香蕉屁屁張晉嘉」後，所有人都說我悶燒QQ~ 不小心露出本性就是這樣QQ... 下午去聽授旗典禮及知能講座，聽完後和毓偉以及浩庭坐在萊爾富附近的桌椅吃東西聊天，聊完就各自回家去了～暑假開始了！！！

# 6/25(六)
今天是頹廢的一天，暑假開始，就是宅在家用電腦玩遊戲之類的，不過還得趕我資工營的講義與簡報，真的是忙死我了！總而言之，禮拜六感覺沒做什麼就過去了～
