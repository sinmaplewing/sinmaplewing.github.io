# 5/22(日)
早上因為太疲累，就沒去主日聚會。中午請聖別弟兄載我到三會所，參加新人成全。參加完後，回家等河水(后羿射月亮)來我家玩。我們玩了一下後，就到漢堡王吃晚餐，吃完就送他回家去了。

# 5/23(一)
早上是離散數學，再來是英文課，我們聽了兩首歌，看了一則新聞，感覺還蠻不錯的XD 中午大夥兒都去本部準備資工之夜，我要留下來上下午的程式設計，然後我就先待在電腦教室玩Portal2，後來下午程式設計的時候，維倫兄就拿我的筆電玩Portal2玩了一整節課XD" 程式設計課結束後，我就搭校車到本部，連絡不上宋教授，我就到英語自學教室去衝時數，結束後跟映竹去德州炸雞吃晚餐，吃完就去看表演！每個表演都超讚的，唯一我覺得最讓我空虛的是，我居然沒跟大家一起準備這個資工之夜的過程，我沒上台表演，也沒幫到甚麼忙，感覺真的很不好ˊˋ 結束後就和映竹、博懷、毓偉、映竹和浩庭去吃食指大丼，吃完後就各自回家了XD

# 5/24(二)
早上是數位邏輯，我依然上的很開心。上完後，就去邵弟兄家的讀經小組，去完又回來分部，和映竹到70315的教官值班室去，我要開始當教官的兩個國小生的家教了！！XD 真特別！XD 一個教的是網頁設計，一個教PowerPoint，還蠻有趣的！XD

# 5/25(三)
早上我整個累癱，結果就沒起來去晨興Orz 直接去上自然探索課，上完後，我就跑去吃可東披薩，吃完就到分部去上程式設計，上完後，晚上聽Ψω試教硬體，聽完後就回家了～

# 5/26(四)
早上是邏輯概論，中午聽威豪試教數邏實作，下午是微積分，晚上吃完飯，本來要聽維倫兄試教，結果又延後一小時，只好先回家做作業XD 還有明天試教的簡報Orz...

# 5/27(五)
早上國文期末考，接著是離散數學交作業，下午是數位邏輯，接著是數邏實驗，然後我又繼續趕簡報，晚上就去試教，結果只有天空龍學長來聽，就變成1對1試教～XD 然後我還跟學長聊到超晚XD

# 5/28(六)
將近睡了整個早上，一整天都待在家裡，趕我的作業簡報...等等之類的東西，既忙又閒的一天XD"
