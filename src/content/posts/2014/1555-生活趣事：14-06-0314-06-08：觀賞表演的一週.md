# 前言
離畢業越來越近，開始越來越有畢業的感覺了，都快痛哭流涕了！QAQ

# 14.06.03－師大吉他社木吉他初級班畢業
![吉他社木吉他初級班畢業](/uploads/2014/06/1535380_729486227073529_6522959124915218147_n.jpg)

木吉他班在十次上課後，終於進入尾聲！在這十堂課學到好多樂理和好多的吉他技巧！

* 感謝仕哲老師這十堂課的教導，以及課後小公園的唱歌！
* 感謝謝欣頻﻿認出我是誰，讓我能用社員價來上初級班！QAQ
* 感謝劉永傑的課後練習，還偷教我們好多東西！XD

大一上剛進來上了六堂課後放棄，大二上進來上了四堂課後放棄，總算在大四下我成功上完了初級班了！痛哭流涕！QAQ！！

晚上為了等皮卡，就在小公園打屁聊天，結果我被放鴿子了QAQ...... 不過也算了～皮卡這幾天實在太累了QAQ~

# 14.06.04－師大吉他社送舊
![吉他社送舊](/uploads/2014/06/10341833_798567403496316_534573662871402214_n.jpg)

雖然我大概有將近兩年沒在吉他社，不過透過送舊來見見我們當年成為理事的幹部們也不錯！跟大家拍照其實還蠻開心的。XD

看著理事的位置又交接給下一屆的感覺，還蠻奇妙的，雖然自己沒有當理事！

P.S. 送舊訂的Pizza還蠻好吃的！

# 14.06.05－師大熱舞社期末舞展
![熱舞社期末舞展](/uploads/2014/06/20140605_120249631_iOS.jpg)

必須要說：是不是每次畢業前都要看一次熱舞社期末舞展，當初高中畢業是如此，如今大學畢業亦是如此。

利用皮卡給我的公關票，讓我坐在了右側第三排的位置，準備看熱舞社期末舞展的表演。由於表演真的太精彩了，無法用言語表達，我只能說不看真的很可惜呀！特別說一下，照片中的那支舞是整次表演帶給大家最詭譎的感覺的一支舞蹈，也是我印象中最深刻的舞蹈，感覺真的怪，但是真的很精彩！如果未來有影片釋出的話，各位可以去看一下！

結束後，就和資工三本柱打哈哈，然後他們去吃快炒，我就在回家路上買了麥當勞回家吃了。

# 14.06.06－師大黃金雨季師駝晚會
![師駝晚會](/uploads/2014/06/WP_20140606_012.jpg)
下午考完第一次太極拳考試，我就直接趕去禮堂參加晚會了！皮卡要去吃他們昨日舞展的慶功宴，所以又沒跟我去禮堂了。再加上Max默默也離開了。QAQ！孤獨。師駝晚會！

其實師駝晚會是社團頒獎的大事，我們師大吉他社也得獎了！開心！其實這次來晚會，主要就是看Bamboo和阿緯表演。晚會中的戲劇很精彩很好笑，舞蹈很讓人懷念當時北友會的時光！XD

晚會結束之後，和Bamboo去古亭麥當勞吃晚餐，我已經連續兩天吃麥當勞啦！！Orz......重點是我們還點超多，如下圖！

![麥當勞晚餐](/uploads/2014/06/WP_20140606_019.jpg)

最後吃完我超撐！

# 14.06.07－PC室做作業
今天與台灣偉和皮卡約去PC室做作業，本想在這之前和皮卡兩人先去公館永淇樂園打街機，但人實在太多了，因此而作罷。

在PC室將Compiler期末作業大致講解一下，然後就回去了！回去之前還跟皮卡兩人去了永淇樂園打了最後一場MaiMai，打完永淇樂園也剛好關了！XD

# 14.06.08－師大資工系送舊
![師大資工系送舊](/uploads/2014/06/10418181_10201108586236149_1773815980218508462_n.jpg)
今日是系上送舊！

剛開始就是一個RPG，而且還用到了老師們和同學們的特色，讓我們回想了好多事情，中間還穿插播放了阿喜老師和蔣老師要給我們的話，好感人！

這四年做過了很多課業上的成就、往我想走的路去做了很多的遊戲；但我也失去了幫忙辦活動、上台表演的機會，尤其是失去了與大家一起辦資工營的回憶，還是讓我耿耿於懷。

RPG結束後，回到了B102，看了懷念的影片後就開始寫卡片，我給了萬能刺蝟神這個群組所有人每人一句話或一個圖，當然我給皮卡的是愛心囉！XD

送舊結束後，和皮卡以及安偉去吃阿里媽媽，吃完去逛了台北地下街，買了明治的起司草莓、打了一場MaiMai後就各自回家了！
