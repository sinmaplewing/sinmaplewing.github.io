# 作者前言
我真的沒想到，一款這麼短的遊戲居然會耗費我快6個月，原因沒有別的，就是懶和忙而已(逃

總而言之，我在這款遊戲上倒是放了不少心血，多少還是會有BUG，還請各位多多回報這樣！先感謝大家的遊玩喔！

**V1.2版**：在隔了一年多後，終於把想要加的功能加上去了，也總算是有點小小的劇情了！希望重新再玩一遍的人，能夠找出更多的樂趣喔！

# 遊戲簡介
灆洢和后羿射月亮想穿越過霍爾特森林，不過灆洢並不會游泳，得利用木板放在河道上才能前進，但是木板放在很深的地方又不太穩，究竟灆洢有沒有辦法通過森林呢？

# V1.2版新增功能
1. **跑步功能：**按下Shift鍵，移動速度可上升。 ![跑步功能](/uploads/2014/08/kai2-21.png)
2. **快速收起木板功能：**按下A鍵，可直接收起同張地圖上的所有木板(除了有東西在上面的木板外)，可以不再使用Q、W鍵慢慢收起木板了。 ![快速收起木板功能](/uploads/2014/08/kai2.png)
3. **劇情任務：**每一關新增了劇情任務，讓破解迷宮的途中還可以看看無厘頭的劇情。![劇情任務](/uploads/2014/08/kai1.png) ![劇情任務](/uploads/2014/08/kai3-1.png) ![劇情任務](/uploads/2014/08/kai2-5.png)
4. **成就系統：**最後看看自己到底有破出些什麼要素吧！ ![成就系統](/uploads/2014/08/kai4-3.png)
